Navigating Advanced Analysis
There are 2 main methods to navigate this page:
Use the table of contents to head to the section and/or subsection of interest. For example, if interested in Black Hole Digestor you may navigate to "Modules-Themselves" and then to the "Black Hole Digestor" section.
Use the browser built in search, usually accessed by pressing "ctrl + F" on PC, or finding your browser app's "Find on page" button for mobile. Type in a key word of interest and click through all associated content.
On mobile, thumbnails should open fully when clicked. On PC, after opening an image you can select "See Full Image" in the Top Left, for better resolution.
Contributing to Advanced Analysis
When contributing to this page, please follow these rules:
Please be sure that the work is still relevant and accurate as to the current released version of the game
Content should show some sort of undocumented analysis, that is not game given info, hence being Advanced Analysis.
Contributions should be in Thumbnail format with the "Wrap text around this item" turned off.
The caption and description, should be in the following format: Author(s) if Known, Quick description, (comma separated list of tags not already in description)
On the Caption please then select the list of tags and change the text type to "Subscript" to make it smaller.
If the content belongs in 2-3 places, please just copy/paste the content w/ caption into the other sections (3 sections max please)
If you need help, have suggestions, or just want someone else to upload content so you don't need to make a Fandom account, give me a ping(not a dm) on discord at Yugiohcd10, or on reddit as u/HubbyLord.
