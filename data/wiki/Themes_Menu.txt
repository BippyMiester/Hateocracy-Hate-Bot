Description
The Theme Menu (commonly called skins) is where players can select tower themes, background themes, choose music and view relics.
Themes
There are 4 main themes: Tower, Background, Menu, and Guardian. Each provide a passive coin bonus even while not active. You are able to randomize themes or select them from the Tower and Background Submenus.
Tower Themes
Changes appearance of the tower. The tower themes can be unlocked in
Milestones
on wave 2,500 of each tier as well as being purchased during
events
. The passive coin multiplier is 0.4% for each owned.
Background Themes
Changes appearance of the background. Background themes can be purchased during events. The passive coin multiplier is 0.8% for each owned.
Menu Themes
Changes appearance of the Menu. Menu themes can be purchased in Guilds. The passive coin multiplier is 0.6% for each owned.
Guardian Themes
Changes appearance of the Guardian. Guardian themes can be purchased in Guilds. The passive coin multiplier is 0.6% for each owned.
Passive Coin Bonus Calculation
The bonuses between themes is additive for a cumulative multiplier. The total multiplier is displayed in the tier multiplier menu. It is calculated as:
Coin Bonus = 1 + 0.004 x Tower Theme Quantity + 0.008 x Background Theme Quantity + 0.006 x Menu Theme Quantity + 0.006 x Guardian Theme Quantity.
